数字・ひらがな 8パズル　Windows 11版
=======================================

【起動方法】
1. ZIPファイルを右クリックします。
2. 「すべて展開」を選びます。
3. 展開されたフォルダーの index.html をダブルクリックします。
4. Microsoft Edge または Google Chrome で数字8パズルが始まります。

画面上部のボタンで、次の2種類を自由に切り替えられます。

・数字8パズル（木製玩具風）
・ひらがな8パズル（メタル調）

同じフォルダーにある hiragana.html を直接ダブルクリックして、
ひらがな版から始めることもできます。

どちらも1ファイルだけで動作し、Python、Node.js、インターネット接続は不要です。
