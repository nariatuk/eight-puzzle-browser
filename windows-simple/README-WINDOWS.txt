木の8パズル　Windows 11かんたん版
====================================

【起動方法】
1. ZIPファイルを右クリックします。
2. 「すべて展開」を選びます。
3. 展開されたフォルダーの index.html をダブルクリックします。
4. Microsoft Edge または Google Chrome でゲームが始まります。

この版は index.html の1ファイルに、画面デザインとゲームプログラムを
すべて内蔵しています。Python、Node.js、インターネット接続は不要です。

画面右上の「Windows 11かんたん版」が、この版の目印です。

※ Windowsから警告が表示された場合
  ZIPの中にある状態ではなく、必ず「すべて展開」してから開いてください。
  index.htmlを右クリックし、「プログラムから開く」からEdgeまたはChromeを
  選ぶこともできます。
